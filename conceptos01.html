Conceptos parte 1

Uso de una etiqueta

Las etiquetas permiten definir muchas propiedades de una página web. Para usarlas debes emparejar adecuadamente su inicio y final de la etiqueta, junto con su contenido que permite añadir características y propiedades a ese contenido en específico.

Resumen de etiquetas

Etiquetas de html5

Etiquetas estructurales

Etiqueta Función
<a> Define un hipervínculo
<article> Establece un artículo
<aside> Define algún contenido relacionado con la página
<body> Define la sección principal del artículo
<br> Produce un salto de línea
<details> Define un widget en que el usuario puede obtener información adicional o controles si él los solicita
<div> Especifica una división de la sección en el documento
<h1> hasta <h6> Define encabezados para poner secciones y subsecciones
<head> Define la sección introductoria de una página web
<header> Representa el encabezado de un documento o sección
<hgroup> Define un grupo de encabezados
<hr> Dibuja una línea horizontal
<html> Define la raíz de un documento html
<footer> Define el pie de página de un documento o sección
<nav> Define un sección de links de navegación
<p> Define un párrafo
<section> Define la sección de un documento como header, footer, etc
<span> Define una sección central sin estilo propio en un documento
<summary> Define un resumen para el elemento <detail>

Etiquetas de metadato

Etiqueta Función
<base> Define el URL base para todos el elementos linkados en el página
<basefront> Especifica la fuente base para un página
<link> Define una relación entre el documento actual y una fuente externa
<meta> Provee metadata estructurado sobre el contenido del documento
<style> Inserta información del estilo (comúnmente CSS) en el encabezado del documento
<title> Define un título para el documento

Etiquetas de forma

Etiqueta Función
<button> Crea un botón clickable
<datalist> Representa un conjunto de opciones predefinidas para un elemento <input>
<fieldset> Especifica un conjunto de campos de forma relacionados
<form > Define un forma HTML para inserción del usuario
<input> Define un control de entrada
<keygen> Es un control que genera un llave private-public
<label> Define una etiqueta para el control <input>
<legend> Define un caption para un elemento <fieldset>
<option> Define un opción en un lista de selección
<select> Define un lista de sección dentro de un forma
<textarea> Define un texto de varios renglones de input control

Etiquetas de formato

Etiqueta Función
<abbr> Define una forma abreviada de un larga palabra o frase
<acronym> Define un acrónimo
<address> Especifica la información de contacto de autor
<b> Define texto en negrita
<bdi> Muestra texto que está aislado de sus alrededores, para formatear texto bidireccional
<bdo> Sobreescribe la actual dirección del texto
<big> Muestra texto en tamaño más grande
<blockquote> Define una cita larga
<center> Centra contenido en el bloque que lo encierra
<cite> Indica una cita o referencia a otro fuente
<code> Formatea texto como código de computadora
<del> Formatea texto como borrador
<dfn> Especifica una definición
<em> Resalta texto
<font> (Obsoleto) Define fuente, color y tamaño de texto
<i> Italiza el texto
<ins> Define un bloque de texto que fue insertado en un documento
<kdb> Especifica texto para un entrada de teclado
<mark> Especifica texto resaltado
<output> Representa el resultado de un cálculo
<pre> Define un bloque de texto preformateado
<progress> Define la compleción de un tarea
<q> Define una cita corta inline
<rp> Provee paréntesis fall-back para buscadores que no soportan anotaciones ruby
<rt> Define la pronunciación de caracteres presentados en una anotación Ruby
<ruby> Representa una anotación Ruby
<samp> Especifica texto como salida muestra desde un programa de computadora
<small> Define texto más pequeño
<strong> Define texto muy enfatizado
<sub> Define texto en subscript
<sup> Define texto en superscript
<tt> Define texto en formato teletipo
<var> Define una variable
<wbr> Define una oportunidad para romper línea

Etiquetas de lista

Etiqueta Función
<dd> Especifica una definición para un término en una lista de definición
<dl> Define una lista de definiciones
<dt> Define un item de la lista de definición
<li> Define un item de la lista
<ol> Define lista ordenada
<menu> Representa una lista de comandos
<ul> Define una lista no ordenada

Etiquetas de tabla

Etiqueta Función
<caption> Define título para tabla
<col> Define valores de atributo para una o más columnas en la tabla
<colgroup> Especifica atributos para columnas múltiples en una tabla
<table> Define una tabla de datos
<tbody> Agrupa un conjunto de filas definiendo el cuerpo central de la tabla de datos
<td> Define una celda de la tabla
<tfoot> Agrupa un conjunto de filas resumiendo las columnas de la tabla
<thead> Agrupa un conjunto de filazs que describe las etiquetas de columnas de la tabla
<th> Define una celda head en la tabla
<tr> Define una fila de celdas en la tabla

Etiquetas de scripting

Etiqueta Función
<noscript> Define contenido alternativo para mostrar cuando el buscador no soporta scripting
<script> Pone script en el documento para procesamento cleint-side

Etiquetas de contenido embebido

Etiqueta Función
<area> Define área específica dentro de un mapa de imágenes
<audio> Incrustra un sonido en el documento html
<canvas> Define un región en el documento, que pode ser usada para dibujar gráficos en el fly vía scripting con JavaScript
<embed> Incrustra una aplicación externa, típicamente multimedia como audio o video
<figcaption> Define un caption o leyenda para una figura
<figure> Representa una figura ilustrada como parte del documento
<frame> Define un marco dentro de un frameset
<frameset> Define una colección de marcos o otro frameset
<iframe> Muestra una URL en un frame inline
<img> Muestra una imagen inline
<map> Define un mapa imagen client-side
<noframes> Define contenido alternativo que muestra en buscadores que no soportan frames
<object> Define un objeto incrustado
<param> Define un parámetro para un objeto o un objeto applet
<source> Define recursos de media alternativos para elementos de media como <audio> o <video>
<time> Define tiempo y/o fecha
<video> Incrusta video en el documento html
