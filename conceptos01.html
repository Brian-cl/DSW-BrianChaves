Conceptos parte 1

Uso de una etiqueta

Las etiquetas permiten definir muchas propiedades de una página web. Para usarlas debes emparejar adecuadamente su inicio y final de la etiqueta, junto con su contenido que permite añadir características y propiedades a ese contenido en específico.

Resumen de etiquetas

Etiquetas de html5

Etiquetas estructurales

Etiqueta Función
<a> Define un hipervíncule
<article> Establece un artícule
<aside> Define algún contenido relacionado con el página
<body> Define la sección principal del artículo
<br> Produce un salto de línea
<details> Define un widget en que el usuario puede obtener información adicional o controles si él los solicita
<div> Especifica una división de la sección en el documento
<h1> hasta <h6> Define encabezados para poner secciones y subsecciones
<head> Define la sección introductoria de una página web
<header> Representa el encabezado de un documento o sección
<hgroup> Define un grupo de encabezados
<hr> Dibuja una línea horizontal
<html> Define el raíz de un documento html
<footer> Define el pie de página de un documento o sección
<nav> Define un sección de links de navegación
<p> Define un párrafo
<section> Define el sección de un documento como header, footer, etc
<span> Define un sección central sin estilo proprio en un documento
<summary> Define un resumen para el elemento <detail>

Etiquetas de metadato

Etiqueta Función
<base> Define el URl base para todos el elementos linkados en el página
<basefront> Especifica el fuente base para un página
<link> Define una relación entre el documento actual y un fuente externa
<meta> Provee metadata estructurado sobre el contenido del documento
<style> Inserta información del estilo (comúnmente CSS) en el encabezado del documento
<title> Define un título para el documento

Etiquetas de forma

Etiqueta Función
<button> Crea un botón clickable
<datalist> Representa un conjunte de opciones predefinidas para un elemento <input>
<fieldset> Especifica un conjunte de campos de forma relacionado
<form > Define un forma HTML para inserción del usuario
<input> Define un control de entrada
<keygen> Es un control que genera un llave private-public
<label> Define una etiqueta para el control <input>
<legend> Define un caption para un elemente <fieldset>
<option> Define un opción en un lista de selección
<select> Define un lista de sección dentro de un forma
<textarea> Define un texto de vario renglones de input control

Etiqueta Función

<acronym> Define un acrónimo

<b> Define texto en negrita
<bdi> Muestra texto que está aislado de sus alrededores, para formatear texto bidireccional

<big> Muestra texto en tamaño más grande
<blockquote> Define un cita larga
<center> Centra contenido en el bloque que lo encierra
<cite> Indica un cita o referencia a otro fuente
<code> Formatea texto como código de computadora
<del> Formatea texto como borrado
<dfn> Especifica un definición
<em> Resalta texto
<font> (Obsolete) Define fuente, color e tamaño de texto
<i> Italiza el texto
<ins> Define un bloque de texto que fue insertado en un documente
<kdb> Especifica texto para un entrada de teclado
<mark> Especifica texto resaltado
<output> Representa el resultado de un cálcule
<pre> Define un bloque de texto preformateado
<progress> Define el compleción de un tarea
<q> Define un corta inline cita
<rp> Provee paréntesis fall-back para buscadores que no soportan anotaciones ruby
<rt> Define el pronunciación de caracteres presentados en un ruby anotación

<tt> Define texto en formato teletypo

Etiquetas de lista

Etiqueta Función
<dd> Especifica un definición para un término en un lista de definición
<dl> Define un lista de definiciones
<dt> Define un item del lista de definición
<li> Define un item del lista

Etiquetas de tabla

Etiqueta Función
<caption> Define títule para tabla
<col> Define valores de atributo para un o más columnes en el tabla
<colgroup> Especifica atributes para columnes múltiples en un tabla

<tbody> Agrupa un conjunto de filas definindo el cuerpo central de el tabla de datos
<td> Define un celda del tabla
<tfoot> Agrupa un conjunto de filas resumindo el columnes del tabla
<thead> Agrupa un cojunto de files que describe el etiquetas de columnes del tabla
<th> Define un celda head en el tabla
<tr> Define un fila de celdas en le tabla

Etiquetas de scripting

Etiqueta Función
<noscript> Define contenido alternativo para mostrar quando el buscador no suporta scripting
<script> Pone script en el documento para procesamento cleint-side

Etiquetas de contenido embebido

Etiqueta Función
<area> Define area specifica dentro de un mapa de imagenes
<audio> Incrusta un sonido en el documento html
<canvas> Define un región in el documente, que pode ser usada para dibujar gráfiques en el fly vía scripting con JavaScript
<embed> Incrusta un aplicación externa, typicamente multimedia como audio o video
<figcaption> Define un caption o legenda para un figura
<figure> Representa un figura ilustrada como parte del documente
<frame> Define un marco dentro de un frameset
<frameset> Define un colección de marcos o otro frameset
<iframe> Muestra un URL en un frame inline
<img> Muestra un imagen inline
<map> Define un mapa imagen client-side
<noframes> Define contenido alternativo que muestra en buscadores que no suportan frames
<object> Define un objecto incrustado
<param> Define un parámetro para un objecto o un objecto applet
<source> Define recursos de media alternativos para elementos de media como <audio> o <video>
<time> Define tiempo e/o fecha
<video> Incrusta video en el documento html
