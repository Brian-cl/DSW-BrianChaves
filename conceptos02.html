Uso de un atributo

Los atributos especifican propiedades de las etiquetas. Por ejemplo a una etiqueta encabezado <h1> se le puede especificar una clase CSS que define unos atributos estéticos al encabezado mediante etiquetas CSS, en atributos propios de HTML se puede especificar altura, background (imagen de fondo), que va acompañado con un valor.

Atributos de Html5

Attributo Opciones Función
accesskey Definidas por el usuario Specifica un atajo de teclado para acceder a un elemento
align right, left, center Alinea etiquetas de forma horizontal
background URL Pone un imagen de fondo detrás de un elemento
bgcolor numeric, hexadecimal, RGB Pone un color de fondo detrás de un elemento
class Definido por le usuario Classifica un elemento para uso con CSS
contenteditable true, false Specifica si el usuario pode editar el contenido de elemento o no
contextmenu Menu id Specifica el menu de contexto para un elemento
data-XXXX Definido por usuario Personaliza attributos. Authores de un documento html pode definir su propios attributos
draggable true, false, auto Specifica le altura
height Valor numérico Especifica le altura de tablas, imagenes o celdas de tablas
hidden escondido Especifica si el elemento deberia ser visible o no
id Definido por usuario Nombra un elemento para uso con CSS
item Lista de elementos Usado para agrupar elementos
itemprop Lista de items Usado para agrupar items
spellcheck true, false Especifica si le elemento debe tener su ortografía o gramática corregida
style CSS style sheet Especifica le inline style de un elemento
subject Usuario define id Especifica le elemento del item correspondente
tabindex Tab número Especifica el orden de un elemento

Valores de html5

Es un attributo de HTML usado para especificar el valor de un elemento con el que es usado. Tiene un significado distinto para diferentes elementos de HTML

Uso

Puede usarse con las etiquetas <input> <button> <meter> <li> <option> <progress> e <param>

  • Cuando se usa con 'button', 'reset' e 'submit' especifica el texto en le botón
  • Cuando se usa con 'text', 'password' e 'hidden' especifica el initial valor de un campo input
  • Cuando se usa con 'checkbox', 'radio' e 'image' especifica el valor associado con le input

Syntaxis

<input value = "value">
