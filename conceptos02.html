Uso de un atributo

Los atributos especifican propiedades de las etiquetas. Por ejemplo a una etiqueta encabezado <h1> se le puede especificar una clase CSS que define unos atributos estéticos al encabezado mediante etiquetas CSS, en atributos propios de HTML se puede especificar altura, background (imagen de fondo), que va acompañado con un valor.

Atributos de Html5

Atributo Opciones Función
accesskey Definidas por el usuario Especifica un atajo de teclado para acceder a un elemento
align right, left, center Alínea etiquetas de forma horizontal
background URL Pone un imagen de fondo detrás de un elemento
bgcolor numeric, hexadecimal, RGB Pone un color de fondo detrás de un elemento
class Definido por el usuario Clasifica un elemento para uso con CSS
contenteditable true, false Especifica si el usuario pode editar el contenido de elemento o no
contextmenu Menu id Especifica el menú de contexto para un elemento
data-XXXX Definido por usuario Personaliza attributos. Autores de un documento html pueden definir su propios atributos
draggable true, false, auto Especifica la altura
height Valor numérico Especifica la altura de tablas, imágenes o celdas de tablas
hidden escondido Especifica si el elemento debería ser visible o no
id Definido por usuario Nombra un elemento para uso con CSS
item Lista de elementos Usado para agrupar elementos
itemprop Lista de items Usado para agrupar items
spellcheck true, false Especifica si el elemento debe tener su ortografía o gramática corregida
style CSS style sheet Especifica el inline style de un elemento
subject Usuario define id Especifica el elemento del item correspondiente
tabindex Tab número Especifica el orden de un elemento

Valores de html5

Es un atributo de HTML usado para especificar el valor de un elemento con el que es usado. Tiene un significado distinto para diferentes elementos de HTML

Uso

Puede usarse con las etiquetas <input> <button> <meter> <li> <option> <progress> e <param>

  • Cuando se usa con 'button', 'reset' e 'submit' especifica el texto en el botón
  • Cuando se usa con 'text', 'password' e 'hidden' especifica el valor inicial de un campo input
  • Cuando se usa con 'checkbox', 'radio' e 'image' especifica el valor asociado con el input

Syntaxis

<input value = "value">
